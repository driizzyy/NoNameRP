Troubleshooting

Can't connect to server?

Make sure you're whitelisted first. If whitelisted, try these steps:

  • Restart FiveM completely
  • Clear FiveM cache (Settings > Clear Cache)
  • Check if your antivirus is blocking FiveM
  • Verify GTA V files in Steam/Epic/Rockstar
  • Contact support on Discord if issue persists
Getting infinite loading screen?

This is usually a caching issue. Try these solutions:

  • Let it load for at least 5 minutes first time
  • Clear FiveM cache and restart
  • Make sure GTA V is fully updated
  • Disable any GTA V mods you may have
  • Check your internet connection
Low FPS or stuttering?

Optimize your game settings:

  • Lower graphics settings in GTA V settings
  • Close background applications
  • Update graphics drivers
  • Increase virtual memory in Windows
  • Play on SSD if possible
FiveM crashes on startup?

Common crash fixes:

  • Run FiveM as administrator
  • Add FiveM to antivirus exceptions
  • Reinstall FiveM completely
  • Verify you have latest Windows updates
  • Check if GTA V launches normally first
