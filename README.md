## Quick Start

### Option 1: Host on GitHub Pages

1. Create a new repository on GitHub named \YourUsername.github.io\
2. Clone the repository locally
3. Copy all files from this folder into the repository
4. Push to GitHub
5. Your site will be live at \https://YourUsername.github.io\

### Option 2: Local Testing

1. Open \index.html\ in your web browser
2. Or use a local server:
   - Python: \python -m http.server 8000\
   - Node.js (http-server): \
px http-server\
3. Visit \http://localhost:8000\

## Customization

### Change Discord Link
Open \index.html\ and find all instances of:
\\\
https://discord.gg/HCdThGFfqN
\\\
Replace with your Discord server invite link.

### Colors
Edit \style.css\ CSS variables:
\\\css
:root {
    --primary-color: #ff4757;
    --secondary-color: #2f3542;
    --accent-color: #0984e3;
}
\\\

### Modify Content
Edit sections in \index.html\:
- Hero section
- Features section
- Careers section
- About section
- Footer
